## 設計支柱

| 支柱 | 說明 |
|------|------|
| 敘事優先 | 故事和角色驅動一切，謎題服務劇情而非阻礙流程 |
| 調查 + 對話 | 點擊場景物件、與 NPC 對話是核心互動方式 |
| 章節制 | 懸疑累積，每章有明確事件與情緒轉折 |
| 暗線流動 | 表面日常之下有更深的秘密逐漸浮現 |
